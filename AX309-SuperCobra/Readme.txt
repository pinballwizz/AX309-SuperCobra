Super Cobra for the AX309 Clone Dev Board.

Notes:
Setup for 2 player arcade controls (6 switches) (Left-Right-Up-Down-Fire-Bomb).
Each switch is connected one side to GPIO Header J2 and other side to VCC 3.3v
Pin Header locations are specified in the "ax309.ucf" File.
Left and Right Audio outputs are on the J2 Header.
See instructions in the "dac.vhd" file for connecting audio to ext amp/spkr.
Coin and Player1 and Player2 Start controls are Keys 1, 2 and 3 buttons on the AX309.

Build:
* Obtain correct roms file for Super Cobra (scobra.zip - see scripts folder for rom details).
* Unzip rom files to scobra folder inside the roms folder.
* Run the make proms script in the scripts folder - adjust filepaths to rom files as necessary.
* Place created VHD Prom files inside the proms folder.
* Open the scobra.xise project file inside the build folder using Xilinx ISE 14.7 webpack or similar and compile.
* Generate Programming file with startup options set to use JTAG Clock.
* Program AX309 Board using Impact and the BIT file generated by XILINX Webpack.
